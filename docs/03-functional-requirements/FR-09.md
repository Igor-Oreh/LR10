# FR-09: Расчет стоимости поездки

## Формулировка требования | Must Have | Средняя

### Бизнес-правила
1. Расчет на основе расстояния (км)
2. Фиксированная ставка за км по тарифу
3. Без динамических коэффициентов спроса
4. Цена фиксируется при подтверждении

**Реализует:** [US-04](../02-user-requirements/US-04.md)
**Зависит от:** [NFR-03](../04-non-functional-requirements/NFR-03.md), [NFR-05](../04-non-functional-requirements/NFR-05.md)

## Критерии приемки
- [ ] Стоимость рассчитывается за ≤ 2 секунды
- [ ] Цена фиксирована и не меняется
- [ ] Нет скрытых наценок

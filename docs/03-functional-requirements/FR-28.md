# FR-28: Активация SOS

## Формулировка требования | Must Have | Высокая

### Бизнес-правила
1. Подтверждение активации (5 сек таймер)
2. Фиксация геолокации и данных поездки
3. Возможность отмены (случайное нажатие)
4. Тихий режим (удержание 3 сек)
5. Точность распознавания голосовой команды ≥85% при уровне шума до 70 дБ, время отклика ≤2 сек, при неуверенном распознавании — запрос подтверждения через интерфейс.

**Реализует:** [US-10](../02-user-requirements/US-10.md)
**Зависит от:** [NFR-01](../04-non-functional-requirements/NFR-01.md)

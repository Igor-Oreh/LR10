# FR-04: Переключение режима интерфейса

## 📜 Формулировка требования

Система должна позволять пользователю переключаться между обычным и крупным режимом интерфейса.

## 🏷 Атрибуты

| Параметр | Значение |
|----------|----------|
| Источник | US-02, KANO F7 |
| Приоритет | Should Have |
| Сложность | Средняя |

### Бизнес-правила
1. Крупный режим: шрифт минимум 18sp, кнопки минимум 48x48 dp
2. Переключение применяется ко всем экранам
3. Настройка сохраняется в профиле
4. Для пользователей 60+ включается автоматически

**Реализует:** [US-02](../02-user-requirements/US-02.md)
**Зависит от:** [NFR-06](../04-non-functional-requirements/NFR-06.md)

## ✅ Критерии приемки
- [ ] Режим переключается без перезагрузки
- [ ] Все элементы масштабируются корректно
- [ ] Настройка сохраняется между сессиями

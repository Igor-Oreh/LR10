# FR-21: Звонок в службу поддержки

## Формулировка требования | Must Have | Низкая

### Бизнес-правила
1. Кнопка доступна на всех экранах
2. Время соединения ≤ 30 секунд
3. Работа 24/7/365
4. Для 60+ кнопка увеличена

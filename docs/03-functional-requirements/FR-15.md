# FR-15: Поиск доступных водителей

## Формулировка требования | Must Have | Высокая

### Бизнес-правила
1. Поиск в радиусе 3 км (расширение до 5 км)
2. Фильтр по тарифу (класс авто)
3. Приоритет: ближайший + высокий рейтинг

**Реализует:** [US-06](../02-user-requirements/US-06.md)
**Зависит от:** [NFR-01](../04-non-functional-requirements/NFR-01.md) - Время поиска ≤ 60 сек

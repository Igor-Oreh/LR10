# FR-01: Отправка SMS с кодом подтверждения

## Формулировка требования

Система должна отправлять SMS с кодом подтверждения на номер телефона пользователя при регистрации и входе.

## Атрибуты

| Параметр | Значение |
|----------|----------|
| Источник | Use Case UC-01 (шаг 2) |
| Приоритет | Must Have |
| Стабильность | Стабильно |
| Сложность | Низкая |

## Детальная спецификация

### Входные данные
- Номер телефона в формате +7 (XXX) XXX-XX-XX
- Тип операции: регистрация / вход

### Бизнес-правила
1. Код состоит из 6 цифр
2. Срок действия кода: 5 минут
3. Повторная отправка доступна через 60 секунд
4. Максимум 3 попытки ввода кода

### Выходные данные
- SMS с кодом отправлено (успех/ошибка)
- Время отправки

### Обработка ошибок
- E1: Номер недействителен → "Неверный формат номера"
- E2: SMS-шлюз недоступен → предложить звонок с кодом
- E3: Превышен лимит отправок → блокировка на 15 минут

## Критерии приемки

- [ ] SMS доставляется в течение 30 секунд
- [ ] Код корректно генерируется и валидируется
- [ ] При неверном коде показывается понятная ошибка
- [ ] Работает повторная отправка через 60 секунд

# FR-05: Автоопределение геолокации

## Формулировка требования

Система должна автоматически определять текущее местоположение пользователя при открытии экрана заказа.

## Атрибуты

| Параметр | Значение |
|----------|----------|
| Источник | Use Case UC-04 (шаг 2), KANO F1 |
| Приоритет | Must Have |
| Сложность | Средняя |

### Бизнес-правила
1. Запрашивается доступ к GPS/Wi-Fi геолокации
2. Точность определения: ±10 метров
3. При успехе: адрес подставляется автоматически
4. При ошибке: предлагается ручной ввод

**Реализует:** [US-03](../02-user-requirements/US-03.md)
**Зависит от:** [NFR-02](../04-non-functional-requirements/NFR-02.md), [NFR-05](../04-non-functional-requirements/NFR-05.md)

## Критерии приемки
- [ ] Местоположение определяется за ≤ 10 секунд
- [ ] Точность ±10 метров
- [ ] При ошибке предлагается ручной ввод

# FR-06: Ручная корректировка адреса

## Формулировка требования

Система должна позволять пользователю вручную ввести или отредактировать адрес подачи и назначения.

## Атрибуты | Must Have | Низкая

### Бизнес-правила
1. Поддержка автодополнения при вводе
2. Валидация адреса (в зоне обслуживания)
3. Обновление точки на карте
4. Перерасчет маршрута и стоимости

**Реализует:** [US-03](../02-user-requirements/US-03.md)

## Критерии приемки
- [ ] Адрес можно ввести вручную
- [ ] Автодополнение работает
- [ ] Маршрут пересчитывается

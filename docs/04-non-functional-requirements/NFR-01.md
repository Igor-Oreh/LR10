# NFR-01: Производительность: Время отклика системы

## Формулировка требования

Система должна обеспечивать время отклика, соответствующее ожиданиям пользователей.

## Атрибуты | Must Have | Производительность

### Метрики производительности

| Операция | Требование |
|----------|------------|
| Открытие приложения | ≤ 2 сек (95-й перцентиль) |
| Расчет стоимости | ≤ 2 сек (95-й перцентиль) |
| Поиск водителя | ≤ 60 сек (80% случаев) |
| Отправка SMS | ≤ 30 сек (95-й перцентиль) |
| Обработка платежа | ≤ 5 сек (95-й перцентиль) |
| Реакция на SOS | ≤ 30 сек (100% случаев) |

### SLA
- Доступность: 99.5% в месяц
- Макс. простой: ≤ 4 часа в месяц
- Автомасштабирование при нагрузке > 70%

## Критерии приемки
- [ ] Все метрики производительности соблюдаются
- [ ] SLA 99.5% подтверждено мониторингом
- [ ] Автомасштабирование работает

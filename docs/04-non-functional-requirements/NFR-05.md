# NFR-05: Интеграция с картографическими сервисами

## Формулировка требования | Must Have | Интеграции

### Провайдеры
1. OpenStreetMap (OSM) - основной
2. Yandex Maps API - резервный
3. Google Maps API - опционально

### Функции
- Геокодинг и reverse геокодинг
- Построение маршрутов
- Расчет ETA с учетом пробок
- Кэширование (TTL: 1 час)
- Fallback при ошибке: последовательность переключения OSM → Yandex Maps → Google Maps. Триггер: получение HTTP 5xx или таймаут ответа >3 сек в течение 2 подряд запросов.

**Применяется к:** FR-05, FR-09, FR-18

## Критерии приемки
- [ ] Геокодинг работает точно
- [ ] Fallback на резервный API
- [ ] Кэширование работает

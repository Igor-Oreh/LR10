# NFR-06: Доступность для пользователей 60+

## Формулировка требования | Must Have | Юзабилити

### Крупный интерфейс
- Шрифт: минимум 18sp
- Кнопки: минимум 48x48 dp
- Контрастность: WCAG 2.1 AA
- Макс. 2 уровня вложенности

### Голосовые функции
- Голосовой ввод адреса
- Голосовая активация SOS
- Озвучка уведомлений

### Поддержка
- Кнопка поддержки на всех экранах
- Пошаговые подсказки

**Применяется к:** FR-03, FR-04, FR-07, FR-21

## Критерии приемки
- [ ] Крупный шрифт ≥ 18sp
- [ ] Кнопки ≥ 48x48 dp
- [ ] Контрастность WCAG 2.1 AA

# NFR-08: Интеграция с внешними сервисами

## 📜 | Must Have | Интеграции

### Сервисы
1. SMS-шлюз (Twilio/локальный)
2. Платежные системы (Apple Pay, Google Pay, СБП)
3. FCM (Android), APNS (iOS)
4. Email-сервис (SendGrid)

### Требования
- Резервные провайдеры
- Health check каждые 5 минут
- Автоматический failover
- Логирование всех вызовов
- SLA: доступность ≥ 99%, ответ ≤ 2 сек

**Применяется к:** FR-01, FR-22, FR-25

## ✅ Критерии приемки
- [ ] Все интеграции работают
- [ ] Failover настроен
- [ ] Мониторинг активен

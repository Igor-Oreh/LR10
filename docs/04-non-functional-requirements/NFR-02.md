# NFR-02: Точность геолокации

## 📜 | Must Have | Точность данных

### Требования
- GPS + Wi-Fi: ±10 метров
- В помещении: ±20 метров
- Частота обновления: каждые 5 сек (активный заказ)
- При точности > 50 метров → предупреждение
- Кэширование последнего известного местоположения

**Применяется к:** FR-05, FR-17, FR-28

## ✅ Критерии приемки
- [ ] Точность ±10 метров в 95% случаев
- [ ] Обновление каждые 5 секунд
- [ ] Корректная работа в помещении

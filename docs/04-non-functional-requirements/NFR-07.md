# NFR-07: Форматы экспорта данных

## 📜 | Should Have | Функциональность

### Форматы
1. PDF (A4, QR-код для проверки)
2. Excel XLSX (структурированные данные, формулы)
3. CSV (UTF-8, разделитель ";")

### Данные для экспорта
- Дата и время поездки, Маршрут, Расстояние, Стоимость
- Способ оплаты, Данные водителя, Реквизиты компании

**Применяется к:** FR-25, FR-26

## ✅ Критерии приемки
- [ ] PDF генерируется корректно
- [ ] Excel содержит все данные
- [ ] CSV импортируется в 1С

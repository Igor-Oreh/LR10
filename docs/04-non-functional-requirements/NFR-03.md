# NFR-03: Безопасность и защита данных

## 📜 | Must Have | Безопасность

### Стандарты
1. PCI DSS Level 1 - платежные данные
2. 152-ФЗ - персональные данные
3. GDPR (международные)

### Шифрование
- В транзите: TLS 1.3
- В покое: AES-256
- Платежные данные: токенизация

### Защита
- JWT токены (24 часа)
- Rate limiting (100 запросов/мин)
- WAF, защита от SQL-инъекций, XSS, CSRF

## ✅ Критерии приемки
- [ ] Соответствие PCI DSS подтверждено
- [ ] Все данные зашифрованы
- [ ] Аудит безопасности пройден

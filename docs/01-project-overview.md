# 📋 Обзор проекта: Мобильное приложение для заказа такси

**Версия:** 1.0  
**Дата:** 11.05.2026  
**Автор:** Орех Игорь Александрович

## 🎯 Бизнес-цели

1. Увеличить количество успешно завершенных заказов на 25% за первые 3 дня активного использования
2. Снизить процент отмены заказов пользователями до <10%
3. Обеспечить покрытие 15% сегмента пользователей 60+ и 25% корпоративного сегмента в пилотном регионе за 6 месяцев
4. Достичь рейтинга приложения не ниже 4.5 в магазинах приложений

## 👥 Целевые персоны

### 1. Студент Игорь (20 лет)
- **Описание:** Ежедневный пользователь, заказывает 1-3 раза в неделю
- **Потребности:** Быстро, дешево, промокоды, сравнение тарифов
- **Устройства:** iOS/Android смартфоны

### 2. Бизнес-аналитик Игорь (38 лет)
- **Описание:** Корпоративный клиент, 3-5 поездок в неделю
- **Потребности:** Предсказуемость, заказ на время, автоматические отчеты
- **Устройства:** Флагманские смартфоны, планшеты

### 3. Пенсионер Игорь Александрович (67 лет)
- **Описание:** Пользователь старшего возраста, 1-2 поездки в неделю
- **Потребности:** Простота, фиксированная цена, крупный интерфейс, наличные
- **Устройства:** Бюджетные Android смартфоны

## 🛠 Основные функции MVP

1. **Автогеолокация** - автоматическое определение местоположения
2. **Фиксация цены** - предварительный расчет без динамических наценок
3. **Отслеживание на карте** - визуализация водителя в реальном времени
4. **Оплата наличными** - возможность оплаты водителю
5. **Кнопка SOS** - экстренная отправка геолокации

## 📊 Приоритеты (MoSCoW)

### Must Have (Обязательно)
- Автогеолокация, фиксация цены, трекинг водителя
- Оплата (карта/наличные), SOS, регистрация
- Базовый заказ такси

### Should Have (Желательно)
- Заказ на время, крупный интерфейс (60+)
- Голосовой ввод, корпоративные отчеты

### Could Have (Возможно)
- Промокоды, кэшбэк, избранное
- Мультиточечные маршруты

### Won't Have (Не в v1)
- Веб-версия, доставка, AI-ценообразование
- Интеграция с 1С, международные платежи

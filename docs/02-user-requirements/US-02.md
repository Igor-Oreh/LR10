# US-02: Настройка профиля (язык, валюта, предпочтения)

## 📜 Формулировка требования

Как пассажир, я хочу настроить свой профиль (язык, валюту, режим интерфейса), чтобы использовать приложение комфортно.

## 👥 Роли пользователей

- Все аутентифицированные пользователи
- Пользователи 60+ (нуждаются в крупном интерфейсе)
- Бизнес-клиенты (нуждаются в выборе валюты отчетности)

## 🏷 Атрибуты

|----------|----------|
| Источник | Use Case UC-01, USER STORY.docx |
| Приоритет | Must Have |
| Стабильность | Стабильно |
| Сложность | Низкая |

## 📋 Детализация по персонам

### Пенсионер Игорь Александрович (67 лет)
**User Story:** Как пожилой человек, я хочу, чтобы приложение сразу открывалось на русском с крупным шрифтом, чтобы не настраивать каждый раз.

**Критерии приемки:**
- [ ] При первом запуске язык по умолчанию - русский
- [ ] Для пользователей 60+ автоматически включается режим "Крупный шрифт"
- [ ] Все кнопки и текст отображаются увеличенными (минимум 18sp)

### Бизнес-аналитик Игорь (38 лет)
**User Story:** Как аналитик, я хочу выбирать валюту отчета (руб/валюта), чтобы удобно вести бухгалтерию.

**Критерии приемки:**
- [ ] В настройках профиля доступен выбор валюты (RUB, USD, EUR)
- [ ] Выбранная валюта применяется к чекам и отчетам
- [ ] Изменения сохраняются между сессиями

- Время настройки профиля ≤ 2 минуты

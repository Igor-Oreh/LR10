# US-11: История поездок и управление профилем

## 📜 Формулировка требования

Как пассажир, я хочу просматривать историю поездок и управлять своим профилем, чтобы контролировать расходы и настройки.

## 📋 Детализация

### Все пользователи
**User Story:** Как пользователь, я хочу видеть список всех моих прошлых поездок с суммами, чтобы отслеживать свои расходы за месяц.

**Критерии приемки:**
- [ ] История поездок доступна в профиле
- [ ] Для каждой поездки указаны: дата, маршрут, сумма, статус
- [ ] Можно отфильтровать по дате
- [ ] Можно повторно заказать поездку из истории

- Время загрузки истории ≤ 2 секунды
- Доступность истории за последние 12 месяцев

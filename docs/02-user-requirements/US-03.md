# US-03: Указание адреса назначения

## 📜 Формулировка требования

Как пассажир, я хочу указать адрес подачи и назначения, чтобы система могла рассчитать маршрут и стоимость.

## 👥 Роли пользователей

- Все аутентифицированные пассажиры
- Студенты (нужна быстрая корректировка)
- Пенсионеры (нужен голосовой ввод и избранное)

## 🏷 Атрибуты

| Параметр | Значение |
|----------|----------|
| Источник | Use Case UC-04 (шаги 2-3), KANO F1,F2 |
| Приоритет | Must Have (KANO: Must-Be) |

| Сложность | Средняя |

## 📋 Детализация по персонам

### Студент Игорь (20 лет)
**User Story:** Как пользователь, я хочу, чтобы геолокация определилась автоматически, но я мог её подправить вручную, чтобы быстро заказать такси.

**Критерии приемки:**
- [ ] При открытии экрана заказа автоматически определяется текущее местоположение
- [ ] Адрес подачи отображается в поле ввода
- [ ] Пользователь может отредактировать адрес вручную
- [ ] При изменении адреса точка на карте обновляется

### Пенсионер Игорь Александрович (67 лет)
**User Story 1:** Как пенсионер, я хочу выбирать адрес из «Избранного» (Дом/Поликлиника) в один клик, чтобы не вводить каждый раз.

**Критерии приемки:**
- [ ] Кнопка "Избранное" отображается на экране выбора адреса
- [ ] При нажатии открывается список сохраненных адресов
- [ ] Выбор адреса происходит одним касанием
- [ ] Система автоматически подставляет адрес и переходит к выбору тарифа

**User Story 2:** Как неопытный пользователь, я хочу использовать голосовой ввод адреса, чтобы не печатать.

**Критерии приемки:**
- [ ] Кнопка микрофона отображается рядом с полем ввода адреса
- [ ] При нажатии активируется голосовой ввод
- [ ] Распознанный текст подставляется в поле адреса
- [ ] Точность распознавания ≥ 90%

## 📊 Метрики успеха

- Время указания адреса ≤ 30 секунд (для избранных ≤ 5 сек)
- Процент успешного автоопределения геолокации ≥ 95%
- Точность распознавания голоса ≥ 90%

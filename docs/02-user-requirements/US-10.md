# US-10: Экстренная помощь (SOS)

## 📜 Формулировка требования

Как пассажир, я хочу иметь возможность вызвать экстренную помощь, чтобы обеспечить свою безопасность.

## 👥 Роли пользователей

- Все пассажиры (особенно важно для 60+)
- Пенсионеры (нужна голосовая активация)

## 🏷 Атрибуты

| Параметр | Значение |
|----------|----------|
| Источник | Use Case UC-22, KANO F10 |
| Приоритет | Must Have (безопасность) |
| Стабильность | Стабильно |
| Сложность | Высокая |

## 📋 Детализация по персонам

### Все пользователи
**User Story:** Как пассажир, я хочу кнопку SOS, которая отправит геолокацию доверенному контакту, чтобы чувствовать себя в безопасности.

**Критерии приемки:**
- [ ] Кнопка SOS доступна на всех экранах
- [ ] При нажатии запрашивается подтверждение
- [ ] Отправляется SMS и Push доверенному контакту
- [ ] Координаты передаются в реальном времени
- [ ] Служба поддержки оповещается автоматически

### Пенсионер Игорь Александрович (67 лет)
**User Story:** Как пожилой человек, я хочу возможность вызвать оператора голосом, если я запутался, чтобы получить помощь.

**Критерии приемки:**
- [ ] Голосовая команда "Помогите" или "SOS" распознается
- [ ] После распознавания запускается сценарий SOS
- [ ] Оператор звонит в течение 30 секунд
- [ ] Включается скрытая аудиозапись

## 📊 Метрики успеха

- Время от активации SOS до звонка оператора ≤ 30 секунд
- Процент успешных отправлений уведомлений ≥ 99.9%
- Доступность функции 24/7/365

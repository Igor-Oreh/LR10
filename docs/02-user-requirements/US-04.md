# US-04: Выбор тарифа и класса автомобиля

## Формулировка требования

Как пассажир, я хочу выбрать тариф (Эконом/Комфорт/Бизнес) и увидеть стоимость, чтобы заказать подходящий автомобиль.

## Роли пользователей

- Все аутентифицированные пассажиры
- Студенты (сравнивают цены)
- Бизнес-клиенты (предпочитают Бизнес класс)
- Пенсионеры (нужна фиксированная цена)

## Атрибуты

| Параметр | Значение |

| Сложность | Низкая |

## Детализация по персонам

### Студент Игорь (20 лет)
**User Story:** Как студент, я хочу видеть сравнение цен (Эконом vs Комфорт) на одном экране, чтобы выбрать оптимальный вариант по соотношению цена-качество.

**Критерии приемки:**
- [ ] Все доступные тарифы отображаются на одном экране
- [ ] Для каждого тарифа видна: цена, время подачи, класс авто
- [ ] Можно быстро переключаться между тарифами
- [ ] Выбранный тариф визуально выделяется

### Бизнес-аналитик Игорь (38 лет)
**User Story:** Как бизнес-клиент, я хочу выбирать «Бизнес» класс по умолчанию, чтобы не выбирать каждый раз.

**Критерии приемки:**
- [ ] В настройках можно установить тариф по умолчанию
- [ ] При открытии заказа автоматически выбирается Бизнес класс
- [ ] Отображаются только доступные для бизнеса опции

### Пенсионер Игорь Александрович (67 лет)
**User Story:** Как пенсионер, я хочу видеть только фиксированную цену без скрытых коэффициентов, чтобы планировать бюджет.

**Критерии приемки:**
- [ ] Цена отображается крупным шрифтом
- [ ] Присутствует пометка "Цена фиксирована"
- [ ] Отсутствуют уведомления о повышенном спросе
- [ ] Цена не меняется после подтверждения

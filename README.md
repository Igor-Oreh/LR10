## 🎯 Краткое описание

Приложение для заказа такси с фокусом на:
- ✅ Прозрачное ценообразование (фиксированная цена)
- ✅ Доступность для пользователей 60+
- ✅ Корпоративный документооборот
- ✅ Безопасность (SOS, трекинг)
